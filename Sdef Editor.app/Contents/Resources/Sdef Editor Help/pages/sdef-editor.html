What is Sdef Editor?

Scripting definition files (sdefs) are XML files that describe everything about an application's scriptability: terminology, implementation information, and complete documentation.
Every scriptable application must provide a definition of its scriptability informationâ€”the terminology available for use in scripts that target the application, as well as the implementation information used to support that terminology. (see Apple documentation and Sdef Format Reference to learn more about Scripting Definition Files).

Sdef Editor allows to simply create, and edit sdefs files. With Sdef editor, you don't have to bother with unreadable XML files.
Sdef Editor provide an easy to use interface to edit and browse an scripting definition.
